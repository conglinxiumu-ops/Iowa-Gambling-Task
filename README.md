# 🃏 THE IOWA TABLE · 爱荷华赌博任务

> 一个用于实验心理学课堂演示的经典决策实验，单文件部署，开箱即用。

---

## 📖 关于本实验

**爱荷华赌博任务（Iowa Gambling Task，IGT）** 由 Bechara、Damasio 等人于 1994 年提出，是研究风险决策的经典范式。实验通过模拟真实赌博情境，观察被试能否在反复试错中学习并趋向长期有利的选择策略。研究范式也常用于考察腹内侧前额叶皮层损伤病人的认知决策功能缺陷。

本程序以赌场牌桌为主题，使用 Art Deco 风格 UI，适合在实验心理学课堂中用于演示 IGT 的核心逻辑，亦可直接用于实际研究，采集被试行为决策数据。

---

## 🎮 实验设计

### 流程

```
欢迎页（输入被试编号）
    ↓
Block 1（50 次选牌）
    ↓
中场休息页
    ↓
Block 2（50 次选牌）
    ↓
结果汇总页（含数据导出）
```

### 牌堆规则

| 牌堆 | 单张奖励 | 损失结构 | 每 10 张净值 | 类型 |
|:----:|:-------:|:--------:|:-----------:|:----:|
| **A** | +$50 | 1 张 -$250（位置随机） | **+$250** | 🟢 优势牌 |
| **B** | +$50 | 5 张小损失，合计 -$250（位置随机） | **+$250** | 🟢 优势牌 |
| **C** | +$100 | 1 张 -$1250（位置随机） | **-$250** | 🔴 劣势牌 |
| **D** | +$100 | 5 张中等损失，合计 -$1250（位置随机） | **-$250** | 🔴 劣势牌 |

> 被试事先不知晓上述规则，需在 100 次选择中自行摸索。

**实验参数：**
- 起始余额：$2,000
- 总试次：100 次（分两个 Block，各 50 次）
- 每副牌共 60 张，每次实验独立随机洗牌（Fisher-Yates 算法）

## ✨ 功能特性

- **零依赖**：单个 `index.html` 文件，仅引用 Google Fonts，无需安装任何依赖
- **被试编号**：欢迎页支持输入被试 ID，自动写入导出数据
- **翻牌动画**：全屏遮罩展示抽到的牌，清晰呈现盈亏结果
- **实时 HUD**：顶部信息栏实时显示余额、试次进度、上轮结果
- **两阶段设计**：Block 1 / Block 2 之间有中场页，可对比前后期策略变化
- **结果汇总**：实验结束后自动生成各牌堆选择频次统计表和条形图
- **CSV 导出**：一键导出完整逐试次数据，包含时间戳（毫秒），可直接用 Excel / R / SPSS / Python 分析
- **键盘快捷键**：支持 `A` `B` `C` `D` 或 `1` `2` `3` `4` 键选牌，翻牌状态下按任意键继续

## 📊 数据格式

点击结果页的"导出数据"按钮，将下载一个 UTF-8（含 BOM）的 CSV 文件，文件名格式为：

```
IGT_<participant_id>_<datetime>.csv
```

**主表字段（每行对应一次选牌）：**

| 字段 | 说明 |
|:-----|:-----|
| `participant_id` | 被试编号 |
| `trial` | 试次序号（1–100） |
| `block` | 所属阶段（1 或 2） |
| `deck` | 选择的牌堆（A / B / C / D） |
| `win` | 本次奖励金额 |
| `loss` | 本次损失金额（0 表示无损失） |
| `net` | 本次净收益（win − loss） |
| `balance_after` | 本次选牌后的余额 |
| `rt_ms_since_start` | 距实验开始的毫秒数（可用于估算反应时） |

CSV 末尾附有按牌堆汇总的选择次数与净值统计，以及最终余额信息。

## 🛠️ 自定义与二次开发

如果您有基本的 HTML 和 JavaScript 知识，可以轻松修改实验参数。用任意文本编辑器打开 `index.html`：

- **修改试次数和余额**：定位到 `<script>` 标签开头的 `TOTAL_TRIALS`、`TRIALS_PER_BLOCK` 和 `START_BALANCE` 常量。
- **修改牌堆收益规则**：定位到 `buildDeckA()` ~ `buildDeckD()` 函数，可以自定义每张牌的 win 和 loss 概率分布。
- **修改主题颜色**：定位到 `<style>` 标签开头的 `:root` CSS 变量区域，可以全局替换主题色。
